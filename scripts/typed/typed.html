EDITING PWNHS SITE

Make sure you edit only the text of the site, located between <p>, <h1>, <h2>, <h3> tags, and images, located in
<img src="./location_of_image.png"/>. Don't touch the styling or JavaScript of the site unless you know what you're doing.

Remember to submit a pull request to the master branch of PWNHS after each of your committs (edits).

If you'd like to change the styling or JS to improve the site, send your proposal in the group chat, open up an issue, or reference the developer docs for help on doing it yourself.
